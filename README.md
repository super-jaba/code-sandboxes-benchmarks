- `multisandbox.py` - run each program in a separate Daytona sandbox
- `singlesandbox.py` - run each program in a single Daytona sandbox
- `multicontext.py` - run each program in a single Daytona sandbox but separate contexts
- `loadbalanced.py` - create a pool of Daytona sandboxes and distribute program executions among them

- `riza.py` - run programs in Riza
